# Running
```
$ python sieve.py

usage: sieve.py [-h] [--sieve SIEVE] [--find-max start end]
                [--benchmark limit iterations]

Sieve of Erastosthenes

options:
  -h, --help            show this help message and exit
  --sieve SIEVE         Sieve of Erastosthenes up to the given integer
  --find-max start end  Tests the maximum value the machine can handle without
                        a MemoryError
  --benchmark limit iterations
                        Run sieve(limit) <iteration> times and prints each run
                        time per line in the end
```
